# 010number 프로젝트

모르는 번호, 스팸 전화, 광고 전화를 검색하고 사용자들이 확인할 수 있는 웹 서비스입니다.

## 📂 디렉토리 구조

html/
 ├─ index.php                ← 요청을 받아 라우팅
 ├─ pages/
 │   ├─ main.php             ← 메인 화면
 │   ├─ phone.php            ← 번호 상세 페이지
 │   └─ register.php         ← 번호 등록 페이지
 ├─ includes/                ← 공통 UI
 │   ├─ head.php
 │   ├─ header.php
 │   └─ footer.php
 ├─ assets/                  ← CSS/JS/이미지
 └─ config.php               ← 공통 설정
